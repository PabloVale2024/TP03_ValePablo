La programación visual es un tipo de lenguaje de programación que permite a sus usuarios ilustrar sus procesos, o dicho de otra forma, se trata de un lenguaje de programación visual que permite a los desarrolladores explicar el proceso con palabras comprensibles para los usuarios, a diferencia de un lenguaje informático tradicional basado en texto. Con un entorno de programación visual como puede ser Dynamo, el desarrollador es capaz de esbozar el flujo de la aplicación para generar una lista de tareas. En el diagrama de flujo se describen las pantallas, las interacciones del usuario y lo que ocurre con la información en cada etapa.
